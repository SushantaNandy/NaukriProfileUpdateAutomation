# LinkedIn Visual Identity Guidelines

## Dimensions
- **Always use a 16:9 aspect ratio** (e.g., 1600x900 or 1920x1080). This format takes up optimal screen real estate on both desktop and mobile feeds without awkward cropping.

## Aesthetic
- Maintain a **Professional tech aesthetic**.
- Use modern, clean visual language (e.g., sleek code editors, abstract data flows, minimalist developer setups).
- Colors should be deep and contrast well, avoiding overly cartoonish or aggressively neon palettes.

## Typography & Details
- **Leverage Text Rendering:** Nano Banana 2 has exceptional text rendering capabilities. Always include the post's main title or a punchy hook directly in the image prompt to create engaging, banner-style visuals.
- Keep the text short (1-5 words) for maximum impact.
- Do not use hyper-realistic human faces to avoid the 'uncanny valley' effect; lean towards conceptual or stylized art if humans are depicted.
